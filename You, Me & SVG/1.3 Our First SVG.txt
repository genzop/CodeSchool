1.3 Our First SVG 
Let's use our new knowledge to create an SVG!

	- Create your first svg element.
	- Specify the SVG's version number by setting the version attribute equal to 1.1.
	- Specify the SVG's namespace by setting the xmlns attribute equal to this URL: http://www.w3.org/2000/svg
	- Set the SVG's viewport attributes using a height of 340 and width of 340.
