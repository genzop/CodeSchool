2.10 Completing the First Column
To complete the first column, we need to add La Amor Peligrosa's record.

	- Add a new row and cell to the content table and include the text 21-0-0.
	- Use this CSS to style the new text: color: #902950; font-family: Georgia, serif; font-size: 20px; font-weight: bold; line-height: 20px; padding-top: 5px;.
